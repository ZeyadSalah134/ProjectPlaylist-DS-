# Audio Player Project

A Qt-based audio player application built with C++ that provides a simple and intuitive interface for playing audio files.

## Features

- Play audio files with an easy-to-use interface
- Built with Qt framework for cross-platform compatibility
- Uses the miniaudio library for audio playback
- Modern and responsive UI design

## Technologies Used

- **C++**: Core programming language
- **Qt Framework**: GUI and application framework
- **miniaudio**: Lightweight audio playback library

## Project Structure

```
AudioPlayer/
├── AudioPlayer/
│   ├── AudioPlayer.cpp       # Main application logic
│   ├── AudioPlayer.h         # Header file
│   ├── AudioPlayer.ui        # Qt UI design file
│   ├── main.cpp              # Application entry point
│   ├── miniaudio.h           # Audio library
│   └── Miniaudio.cpp         # Audio implementation
├── AudioPlayer.slnx          # Visual Studio solution file
└── .gitignore
```

## Requirements

- Visual Studio 2019 or later
- Qt Framework (5.x or 6.x)
- Windows OS (currently configured for Windows development)

## Building the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/ZeyadSalah134/ProjectPlaylist-DS-.git
   ```

2. Open `AudioPlayer.slnx` in Visual Studio

3. Configure Qt paths in your Visual Studio environment

4. Build and run the project

## Usage

1. Launch the application
2. Use the interface to load audio files
3. Control playback using the provided buttons
